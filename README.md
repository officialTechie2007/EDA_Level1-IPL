# 📊 Exploratory Data Analysis (EDA) Projects

A collection of beginner-friendly **Exploratory Data Analysis (EDA)** projects built using **Python, Pandas, Matplotlib, and Seaborn**. The objective of this repository is to understand real-world datasets, clean and preprocess data, discover meaningful insights, and visualize relationships between different features.

## 🚀 Project Included

### 🏏 IPL Complete Dataset (2008–2025)

**Dataset:** IPL Complete Dataset (2008–2025 Enhanced Edition)

This project explores various aspects of the Indian Premier League, including teams, players, matches, and performance trends through statistical analysis and data visualization.

**Key EDA Steps**

* Dataset overview
* Statistical summary
* Missing value analysis
* Data cleaning
* Correlation analysis
* Distribution analysis
* Pair plots and scatter plots
* Box plots for feature comparison

## 📈 EDA Workflow

The notebooks follow a simple three-step workflow:

1. **Understand the Data**

   * Dataset preview
   * Shape
   * Column information
   * Statistical summary
   * Unique values

2. **Clean the Data**

   * Handle missing values
   * Remove redundant columns
   * Prepare data for analysis

3. **Analyze the Data**

   * Correlation Heatmap
   * Pair Plot
   * Scatter Plot
   * Histogram
   * Box Plot
